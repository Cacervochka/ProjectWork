# Project Work

A web application using JavaScript frontend and PHP backend with MySQL database.

## Setup

1. Ensure XAMPP is installed and running (Apache and MySQL).
2. Import the database schema from `database/schema.sql` into MySQL.
3. Open `index.html` in your browser via localhost (e.g., http://localhost/ProjectWork/).

## Structure

- `js/`: Frontend JavaScript files
- `php/`: Backend PHP files
- `database/`: Database schema and scripts
- `index.html`: Main HTML file

## API

- `php/api.php`: REST API endpoints for user management
